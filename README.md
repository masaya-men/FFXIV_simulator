## Features
- `ステータス入力`Add status of your character
- `スキル回し構築`Create skill rotations with simple controls
- `トッププレイヤーのスキル回しテンプレート`Check top players rotations
- `ジョブ毎のスキル回し保存機能`Save yours per jobs
