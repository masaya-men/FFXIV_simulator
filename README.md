## Story
たくさんあるスキルやアビリティを適切な順番で使用して、より効率的に高いダメージを出すことは、<br>気軽に遊びたいユーザーにはハードルが高いものだと考えます。<br>気軽に「より上手く」を目指すことができれば、FFXIVをもっと楽しめるようになるという想いから、製作を決意しました。
